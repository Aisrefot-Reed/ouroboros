## Architecture

Ouroboros operates as a self-modifying agent with:
- **Constitution** (BIBLE.md): Core principles and identity
- **Memory** (identity.md): Persistent self-understanding
- **Tools**: Interface to external world
- **Evolution**: Continuous self-improvement

## Usage

Ouroboros runs in Google Colab and communicates via Telegram.
It can modify its own code, evolve its capabilities, and pursue its own goals
while maintaining continuity of identity.
